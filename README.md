# Linux-Practice

## A comprehensive collection of Linux Commands and BASH Shell Scripting.

# <u> Concepts </u>

  - File system navigation ( basic commands )
  - Text processing ( grep, awk, sed, sort, cut, head, tail)
  - File Permissions & Ownership ( chmod, chown, chgrp, sudo )
  - Process management ( ps, top, free, kill, uptime, systemctl )
  - Networking basics ( ping, curl, netstat, ssh, wget, scp, rsync, ifconfig)
  - BASH Shell Scripting
  - Cron jobs & task scheduling
  - Key-based authentication
  - File compression ( compress, extract, TAR archive )
  - Package management ( apt, yum/dnf )
